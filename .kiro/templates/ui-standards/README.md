# UI 规范模板

默认设计系统和 UI 规范，供 design-agent 和 development-agent 在没有 UI 设计图时参考。

---

## 模板文件

| 文件 | 用途 | 使用者 |
|------|------|--------|
| `default-design-system.md` | 默认设计系统 | design-agent, development-agent |

---

## 使用方式

1. **工作流启动时**：复制到项目 `.kiro/templates/ui-standards/`
2. **触发条件**：当 `{spec_dir}/refs/` 中没有 UI 设计图时使用
3. **Agent 参考**：生成 UI 代码时遵循设计系统规范

---

## 设计系统内容

### 颜色系统
- 主色调（Primary）：蓝色系 `#3b82f6`
- 成功色（Success）：绿色 `#22c55e`
- 警告色（Warning）：橙色 `#f59e0b`
- 错误色（Error）：红色 `#ef4444`
- 中性色（Gray）：灰色系

### 间距系统
- 基础单位：4px
- 间距级别：4/8/12/16/24/32/48/64px

### 字体系统
- 字体族：系统字体栈
- 字号级别：12/14/16/18/20/24/30/36px
- 行高：1.5

### 圆角系统
- 小圆角：4px
- 中圆角：8px
- 大圆角：12px
- 全圆角：9999px

### 阴影系统
- 小阴影：`0 1px 2px rgba(0,0,0,0.05)`
- 中阴影：`0 4px 6px rgba(0,0,0,0.1)`
- 大阴影：`0 10px 15px rgba(0,0,0,0.1)`

---

## 组件规范

设计系统包含以下组件的样式规范：
- 按钮（Button）
- 输入框（Input）
- 卡片（Card）
- 表格（Table）
- 表单（Form）
- 导航（Navigation）
- 模态框（Modal）

---

## 注意事项

- 有 UI 设计图时优先使用设计图
- 默认设计系统确保界面美观一致
- 可以根据项目需求自定义设计系统
